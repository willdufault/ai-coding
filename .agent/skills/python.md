**Syntax:**
- Use absolute imports, not relative imports

**Formatting:**
- Add a concise docstring to the top of each Python file explaining its purpose

**Type hints:**
- Only add type hints for function parameters and return values
- Use guard statements to avoid static type checker errors

**Tools:**
- Use `uv` for package management (ex: `uv add requests`)
